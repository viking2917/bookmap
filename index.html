Historical Maps using MapLibre and Open Historical Maps

Live maps

  • map.html - A simple map with a "bookish" style
  • map_openhistorical.html - The same style except incorporates Open Historical Maps data layers, to include countries/kingdoms. The map year in my example is set at 1187 AD.

Styles

  • bookmap.json - Basic style file, used by map.html
  • bookmap_openhistoricalmap.json - Style file that includes data from Open Historical Maps, used in map_openhistorical.html

Map Sprite Set

Available sprites for use in maps.

the script 'npm run sprites' packages up a set of image files as a sprite file for use in a style. In your style json file, you will reference it something like: 
{
  "sprite": "https://viking2917.github.io/bookmap/sprites/bookmap_spritesheet",
}

Sprite Files

  • bookmap_spritesheet.json - Sprite metadata
  • bookmap_spritesheet.png - Sprite image (1x)
  • bookmap_spritesheet@2x.png - Sprite image (1x)
